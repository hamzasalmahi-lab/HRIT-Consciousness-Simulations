# HRIT-Consciousness-Simulations

This repository contains the official Python simulations for **Hierarchical Recursive Integration Theory (HRIT)**, a unified neurobiological framework of consciousness. These scripts provide the computational evidence for the transitions between phenomenal tiers and the formal stability metrics (RSI/VPM) proposed in the HRIT manuscript.

## Overview
HRIT posits that consciousness is a non-linear product of four specific neural components:
1. **HGM** (Hierarchical Generative Modeling)
2. **CS** (Counterfactual Simulation)
3. **AFEM** (Active Free Energy Minimization)
4. **RSR** (Recursive Self-Representation)

The core mathematical claim is $C = f(\prod Z_i)$, where consciousness ($C$) is a multiplicative function of these components.

## Key Simulations & Figures

To reproduce the specific figures from the HRIT manuscript, run the corresponding scripts listed below:

### 1. The VPM Principle (Variance-Precedes-Mean)
**Files:** `VPM Time-Series, Variance Precedes Mean.py`, `The VPM Principle (Early Warning Simulation).py`, `Stochastic VPM (Critical Slowing Down).py`
* **Insight:** These scripts simulate the **Critical Slowing Down** of the Recursive Stability Index (RSI). They demonstrate how a spike in signal variance (the VPM signature) serves as an early-warning signal for a "dissociative snap," long before the mean stability drops.



### 2. Stability Phenotyping (Clinical Mapping)
**Files:** `Stability Phenotyping Radar Chart.py`, `Clinical Radar Plot Component-Specific Failure.py`, `Figure 8 — Neuromodulatory Fingerprints (Radar).py`
* **Insight:** These simulations use polar projections to map clinical disorders (DPDR, Amnesia, Cotard’s) onto the HRIT component space, illustrating how selective degradation of specific recursive loops creates distinct phenomenal "footprints."



### 3. The Multiplicative Snap vs. Additive Fade
**Files:** `The Multiplicative Snap vs. Additive Fade.py`, `The Product Equation The Necessary & Sufficient Visual.py`, `Integration-Weighted Scaling ($f_3$).py`
* **Insight:** A visualization of the "Cliff Effect." Unlike additive models where consciousness dims linearly, the HRIT product structure explains why consciousness appears as a discrete, all-or-nothing "snap" into coherence.



### 4. Developmental & State Profiles
**Files:** `RDI × RSI Developmental Trajectory.py`, `Sleep Stage Consciousness Profile.py`
